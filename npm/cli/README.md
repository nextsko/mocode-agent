# @mocode/cli

Terminal AI coding assistant — install mocode via npm.

## Install

```sh
npm install -g @mocode/cli
# or one-shot
npx @mocode/cli
```

## Usage

```sh
mocode          # start interactive TUI
mocode --help
```

## Supported Platforms

| OS | Arch |
|---|---|
| Linux | x64, arm64 |
| macOS | x64 (Intel), arm64 (Apple Silicon) |
| Windows | x64 |

Binary is downloaded from [GitHub Releases](https://github.com/nextsko/mocode-agent/releases) at install time. No compilation required.
